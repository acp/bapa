# bapa

an experiment on text parsing & whatsapp integration
version : 1.1.2

# how 2 use

reading :
```shell
node index.js -r
```

writing :
```shell
node index.js -w 1,2,3
```

listing chapter :
```shell
node index.js -e Kej # kej is short for kejadian (indonesian for genesis)
```
